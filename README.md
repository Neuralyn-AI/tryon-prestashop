## What is Neuralyn TRYON?

Neuralyn TRYON is a **virtual fitting room** system that allows your customers to try on clothes virtually before purchasing. Using advanced AI technology, customers can see how clothing items look on them without leaving your online store.

This PrestaShop module integrates the Neuralyn TRYON widget into your store, enabling virtual try-on functionality on product pages.

## See it in Action

Experience the virtual try-on feature on our demo store:

**[https://demo-prestashop.neuralyn.com.br/](https://demo-prestashop.neuralyn.com.br/)**

Simply navigate to any clothing product page and click the "Try it on" button.
